# Needed python packages

```shell
# main computation script
pip3 install numpy

# exact diagonalization
pip3 install scipy

# plotting features
pip3 install matplotlib

# computational helper pre-calculation scripts
pip3 install sympy
pip3 install PyQt6
pip3 install Pillow
```
